# 🚗 VIN-код для системи управління СТО

## Швидкий старт

### Що це дає?
- ✅ Введення VIN-коду при створенні замовлення
- ✅ Розшифрування VIN з інформацією про авто (марка, модель, рік)
- ✅ Перегляд історії обслуговування для кожного автомобіля
- ✅ Валідація VIN-коду за міжнародними стандартами
- ✅ VIN-код автоматично зберігається у автомобілі

---

## 📍 Де це знаходиться в системі?

### 1. **Dashboard → Нове замовлення**
Крок 1 (Авто та Клієнт) має нове поле:
```
VIN-КОД (опціонально):
[Введіть VIN-код автомобіля]
Макс. 17 символів
```

### 2. **Таблиця замовлень**
Нова колонка **VIN-код** показує:
- Скорочений VIN з правою послугу (блакитний фон)
- Або "—" якщо не вказан

### 3. **Сторінка Customers**
На кожній картці клієнта під номером авто з'являється:
```
VIN: JT2BF22K3... (натисніть для перегляду історії)
```

### 4. **Деталі клієнта**
При розкритті детальної інформації про автомобіль:
```
Toyota Camry
Номер: АА1234ББ
VIN: JT2BF22K3M0000001 (натисніть для перегляду історії)
```

---

## 🔍 Як користуватися?

### Варіант 1: При створенні замовлення
```
1. Dashboard → "+ Нове замовлення"
2. Заповніть: Клієнт, номер авто
3. (ОПЦІОНАЛЬНИЙ) Введіть VIN-код в поле
4. Виберіть послуги
5. Збережіть замовлення
```

### Варіант 2: Перегляд історії VIN
```
1. Перейти на Customers
2. Натиснути на картку клієнта
3. Натиснути на VIN-код у списку автомобілів
4. Відкриється модальне вікно з:
   - Валідацією VIN
   - Інформацією про авто (марка, модель, рік)
   - Історією обслуговування
```

## 📋 Технічні деталі

### API Endpoints (для розробників)

```bash
# Розшифрування VIN
POST /api/vin/decode
{ "vin": "JT2BF22K3M0000001" }

# Валідація VIN
POST /api/vin/validate
{ "vin": "JT2BF22K3M0000001" }

# Історія обслуговування
GET /api/vin/history/:vin

# Повна перевірка (валідація + decode + історія)
POST /api/vin/check
{ "vin": "JT2BF22K3M0000001" }
```

### Структура даних

**VIN зберігається у двох місцях:**

1. **У замовленні (Order)**
   ```javascript
   order.vinCode = "JT2BF22K3M0000001"
   ```

2. **У автомобілі (Car)**
   ```javascript
   car.vin = "JT2BF22K3M0000001"
   ```

## 🐛 Хто помогті?

### Розповсюджена проблема: "VIN-код не зберігається"
**Рішення:** Переконайтесь, що база даних мігрована:
```bash
cd sto-backend
npx prisma migrate deploy
```

### Розповсюджена проблема: "API перевірки недоступна"
**Рішення:** Перевірте, що бекенд запущено на :5000:
```bash
# У терміналі 1:
cd sto-backend && npm start

# У терміналі 2:
cd ./ && npm start
```
